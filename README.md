# mini-rbac-serve
[mini-rbac](https://github.com/zy7y/mini-rbac)，以上述项目进行部分内容改变，用作笔记配套的源码
- JS 替换为 TS
- SQLite 替换为 MySQL
- FastAPI 缓存替换为 Redis
